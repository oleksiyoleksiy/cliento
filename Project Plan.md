# Project Plan
## Проєкт: Сайт для фрилансерів і замовників

## 1. Мета проєкту
Створення простої вебплатформи для взаємодії замовників і фрилансерів
через публікацію проєктів та подання заявок.

## 2. Технологічний стек
- Frontend: React, HTML, CSS (Tailwind)
- Backend: PHP (Laravel)
- Database: MySQL

## 3. Основні ролі
- Замовник
- Фрилансер

## 4. План розробки (Sprint 1)
- Аналіз вимог і архітектури
- Проєктування бази даних
- Реєстрація та авторизація
- Створення і перегляд проєктів
